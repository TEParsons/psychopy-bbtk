TPadVoiceKey
-------------------------------

To import TPadVoiceKey, you can either use::

    from psychopy_bbtk.tpad import TPadVoiceKey

or, any time after `psychopy.plugins.activatePlugins` has been called::
    from psychopy.hardware.bbtk import TPadVoiceKey
