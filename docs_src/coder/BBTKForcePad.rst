BBTKForcePad
-------------------------------

To import BBTKForcePad, you can either use::

    from psychopy_bbtk.forcePad import BBTKForcePad

or, any time after `psychopy.plugins.activatePlugins` has been called::
    from psychopy.hardware.bbtk import BBTKForcePad
